APRENDA MARTE EXPLORAÇÃO

Marte, o quarto plano a partir do Sol, é poeirento, frio e mundo desértico com uma atmosfera muito distinta. O objetivo do programa de exploração de Marte é fornecer uma fluxo contínuo de informações e descobertas científicas através de uma série de órbitas robóticas cuidadosamente selecionadas.
